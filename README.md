# Exploratory Data Analysis (EDA) Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns, trends, and relationships among variables. Various statistical summaries and visualizations are used to understand the data and identify key influencing factors.

## Objectives

* Explore and understand the dataset.
* Generate statistical summaries.
* Identify missing values and duplicate records.
* Analyze relationships between variables.
* Visualize data using graphs and heatmaps.
* Extract meaningful insights from the dataset.

## Dataset

* **Dataset:** Titanic Dataset
* **File Name:** `titanic.csv`

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
